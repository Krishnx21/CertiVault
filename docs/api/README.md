# CertiVault API Documentation

Full API reference is documented inline in the main [README](../../README.md#api-reference).

## Endpoints Overview

- **Authentication** — `/api/auth/*`
- **Documents** — `/api/documents/*`
- **Verifications** — `/api/verifications/*`
- **Vault Members** — `/api/vault/*`
- **Notifications** — `/api/notifications/*`
- **Search** — `/api/search`
- **Dashboard** — `/api/dashboard/*`
- **Health** — `/health/live`, `/health/ready`
- **Metrics** — `/metrics`
